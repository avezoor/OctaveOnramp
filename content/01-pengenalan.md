# Pertemuan 1 - Pengenalan Octave

## Tujuan
Mahasiswa mengenal GNU Octave, command window, dan dokumentasi dasar.

## Materi
- Apa itu Octave
- Cara menjalankan perintah
- `help`, `lookfor`, dan dokumentasi resmi

## Praktik
```octave
2 + 3
a = 10
b = 5
a * b
```

## Latihan
1. Jalankan operasi aritmatika sederhana di command window.
2. Buat dua variabel angka, lalu tampilkan hasil penjumlahan dan perkaliannya.
3. Gunakan `help disp` untuk membaca dokumentasi fungsi `disp`.
